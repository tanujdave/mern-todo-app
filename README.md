## Todo App
### Prerequisites

* node
* npm
* mongodb (Already setup on mongodb atlas)

1. Clone this repository

2. Install server dependencies
    ```bash
    $ cd server
    $ npm install
    ```
3. Install client dependencies
    ```bash
    $ cd client
    $ npm install
    ```

## Run the app

1. Start the server
    ```bash
    $ cd server
    $ npm start
    ```
2. Start the client
    ```bash
    $ cd client
    $ yarn start
    ```
4. Browse to `http://localhost:3000/`
